PILIHAN 1: INSTALLER LENGKAP (Recommended)
===========================================

1. Right-click file:  INSTALL.bat

2. Pilih: "Run as administrator"

3. Ikuti step-by-step installer

4. Setelah selesai, buka: http://pos_griya.test/test-storage.php

5. Pastikan semua check: YES ✅

6. Coba upload gambar


JIKA SYMBOLIC LINK GAGAL (Muncul WARNING):
==========================================

1. Right-click file:  FIX-STORAGE-LINK.bat

2. Pilih: "Run as administrator"

3. Klik "Yes" pada UAC prompt

4. Test lagi: http://pos_griya.test/test-storage.php


SOLUSI TERAKHIR (Jika masih gagal):
===================================

1. Enable Windows Developer Mode:
   - Buka Settings
   - Privacy & Security > For developers
   - Aktifkan "Developer Mode"
   - Restart komputer
   - Jalankan FIX-STORAGE-LINK.bat lagi

2. Atau gunakan copy method:
   - Jalankan: USE-COPY-STORAGE.bat
   - Tidak ideal tapi akan bekerja


PILIHAN 2: QUICK SETUP
======================

1. Right-click file:  setup-laragon.bat

2. Pilih: "Run as administrator"


3. Kirim ke developer


FILE-FILE PENTING:
==================

• setup-laragon.bat          → Script setup otomatis
• diagnostic-laragon.bat     → Script cek masalah
• SETUP_LARAGON.md           → Dokumentasi lengkap
• test-storage.php (public/) → Test permission di browser


CATATAN:
========

Masalah ini HANYA terjadi saat deploy ke Laragon.
Dengan "php artisan serve" tidak ada masalah.

Penyebabnya:
- Folder livewire-tmp tidak ada
- Symbolic link format Unix tidak bekerja di Windows
- Permission storage belum di-set

Setup script sudah memperbaiki semua masalah di atas.
